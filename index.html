Welcome to ZyberCraft

Play Minecraft in your browser - no downloads required!

Server Information

ZyberCraft is a Minecraft server that runs directly in your web browser using Eaglercraft technology.

You can play from anywhere - school, work, or home - without installing anything!

How to Connect

  1. Visit an Eaglercraft client website (like eaglercraft.com)
  2. Enter the following server address:
wss://your-replit-url/zybercraft

Replace "your-replit-url" with your actual Replit URL.

Server Rules

  • Be respectful to other players
  • No griefing or stealing
  • Have fun!
